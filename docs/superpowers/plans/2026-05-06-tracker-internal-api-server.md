# Tracker Internal API + Server Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `tracker/internal/api` and `tracker/internal/server` — the server-side wire-protocol surface that the merged `plugin/internal/trackerclient` already speaks. Nine unary RPCs (5 real, 4 Scope-2 stubs), two server-initiated push-stream initiators (offer + settlement), heartbeat reader, mTLS+SPKI termination, graceful shutdown, and a `token-bay-tracker run` cobra subcommand to wire everything up.

**Architecture:** `internal/server` owns transport (QUIC), mTLS termination, per-connection state, the accept loop, the heartbeat reader, and the push-stream initiators. `internal/api` owns the dispatch surface: a `Router` mapping `RpcMethod → handler`, one file per RPC, narrow per-handler interfaces, plus encoders/decoders for push messages. Hand-wired DI via a `Deps` struct on each constructor — nil subsystem fields auto-stub their handlers to `RPC_STATUS_INTERNAL "NOT_IMPLEMENTED"` (Scope-2 contract). Every QUIC/TLS detail mirrors the as-merged `plugin/internal/trackerclient` exactly so real plugin clients interop without changes.

**Tech Stack:** Go 1.25, `google.golang.org/protobuf`, `github.com/quic-go/quic-go` (new for tracker), stdlib `crypto/tls` + `crypto/x509` + `crypto/ed25519`, `github.com/google/uuid` (new for tracker), `github.com/spf13/cobra`, `github.com/rs/zerolog`, `github.com/stretchr/testify`.

## Wire-reality reconciliation (read first)

Three places where the as-merged plugin client diverges from the spec — the server matches the **client's actual wire**, not the spec's prose. The plan below encodes the wire reality. Each is a one-line plan adjustment, captured here so reviewers don't bisect against the spec:

1. **Heartbeat-stream framing.** Spec §5.2 / trackerclient spec §6.2 said the heartbeat stream's first frame is `RpcRequest{method=RPC_METHOD_UNSPECIFIED, payload=HeartbeatPing}`. The merged client (`plugin/internal/trackerclient/heartbeat.go:65`) writes a bare `HeartbeatPing` proto, no `RpcRequest` wrapper. Server reads raw `HeartbeatPing` on the heartbeat stream and replies with raw `HeartbeatPong`. The "method-zero is the heartbeat-channel marker" rule survives only as a validator check on regular RPC streams (rejects method=0 there).
2. **Push-stream class tag.** Server-initiated streams begin with a single-byte class tag — `0x01` = offer, `0x02` = settlement — written *before* the framed push proto (`plugin/internal/trackerclient/push_settlements.go:13-16`). Spec didn't mention this; client code is the source of truth.
3. **Heartbeat-stream identification.** "First client-initiated bidi stream after handshake" is positional only. The server-side connection owner counts client-initiated streams; the first becomes the heartbeat stream regardless of frame content (peeking would require buffering and racing the first ping). Stream-id ordering is monotonic in `quic-go`; the lowest-numbered client bidi stream is always the heartbeat.

## Task ordering rationale

1. **Phase 0 (preconditions).** Confirm `shared/proto/rpc.proto` is on `main` and that `RPC_METHOD_*`, `RpcRequest`, `RpcResponse`, `OfferPush`, `OfferDecision`, `SettlementPush`, `SettleAck`, `HeartbeatPing`, `HeartbeatPong`, `Validate*` are all importable. If any is missing, abort and surface the gap before touching the tracker.
2. **Phase 1 — config additions.** Grow `ServerConfig` with the four new fields + defaults + validators + `testdata/full.yaml` update. Lands as one commit; downstream phases compile against it. Cross-cutting only within `tracker/`.
3. **Phase 2 — `internal/api` foundation (no transport).** `errors.go`, `RequestCtx`, `Router` skeleton, `Dispatch` shell, `PushAPI`. Pure Go, zero net imports. All 9 RPCs land as `ErrNotImplemented` stubs at this point. `internal/api` compiles + ships green tests with **no `internal/server` work yet**.
4. **Phase 3 — real handlers.** Replace the four Scope-2 reals (`balance`, `advertise`, `stun_allocate`, `turn_relay_open`) and `enroll` with handler logic. Each is a single source file + test against a fake service literal + golden bytes. Stubs (`broker_request`, `settle`, `usage_report`, `transfer_request`) keep `ErrNotImplemented` and ship a verification test.
5. **Phase 4 — `internal/server` foundation.** TLS config + SPKI extraction, length-prefix framing helper, `*Connection` skeleton, `Server` skeleton, `New` validation. Listener and accept loop *not* yet wired. Pure unit tests with stubs.
6. **Phase 5 — listener + accept loop.** Bring up the QUIC listener; accept inbound connections; spawn `*Connection`. Integration with `*Connection` is still feature-flagged off (no streams handled yet).
7. **Phase 6 — heartbeat reader.** Wire the first-client-stream branch. Once it lands, a real plugin client connects + heartbeats + appears in `PeerCount`.
8. **Phase 7 — RPC stream acceptor.** Wire the >1 stream branch through `api.Router.Dispatch`. End of phase: every real handler is reachable from a client.
9. **Phase 8 — push initiators.** `Server.PushOfferTo` + `Server.PushSettlementTo` + per-stream owner goroutines. Component-tier tests now drive offer/settlement round-trips.
10. **Phase 9 — shutdown.** Drain logic + idempotency + outstanding-channel reject delivery.
11. **Phase 10 — `cmd run` subcommand.** Composition root: load config, build subsystems, construct `Deps`, call `Run`, install signal handler.
12. **Phase 11 — component tier.** `tracker/test/fakeclient` + `tracker/test/component`. Wire round-trips without UDP using the loopback transport copied from the plugin.
13. **Phase 12 — integration tier (build-tagged).** Real `quic-go`, real mTLS, real UDP. Four files, copy-pasted helpers from §10.3 of the spec verbatim.
14. **Phase 13 — cleanup.** Delete `tracker/internal/session/`, update `tracker/CLAUDE.md` "Project layout" if needed (it already lists api+server+session — drop session). Run `make check` repo-wide.

Each phase is one commit per task. Tasks within a phase are TDD red→green→refactor cycles. New runtime deps (`github.com/quic-go/quic-go`, `github.com/google/uuid`, `golang.org/x/sync`) are folded into the phase that first imports each — no separate "add deps" task.

```go
package api

import (
	"errors"

	tbproto "github.com/token-bay/token-bay/shared/proto"
)

// Sentinel error type. Handlers construct one via Err* helpers; Dispatch
// converts via ErrToResponse.
type apiErr struct {
	status tbproto.RpcStatus
	code   string
	msg    string
}

func (e *apiErr) Error() string { return e.code + ": " + e.msg }

func ErrInvalid(msg string) error {
	return &apiErr{status: tbproto.RpcStatus_RPC_STATUS_INVALID, code: "INVALID", msg: msg}
}

func ErrNoCapacity(msg string) error {
	return &apiErr{status: tbproto.RpcStatus_RPC_STATUS_NO_CAPACITY, code: "NO_CAPACITY", msg: msg}
}

func ErrFrozen(msg string) error {
	return &apiErr{status: tbproto.RpcStatus_RPC_STATUS_FROZEN, code: "FROZEN", msg: msg}
}

func ErrNotFound(msg string) error {
	return &apiErr{status: tbproto.RpcStatus_RPC_STATUS_NOT_FOUND, code: "NOT_FOUND", msg: msg}
}

func ErrUnauthenticated(msg string) error {
	return &apiErr{status: tbproto.RpcStatus_RPC_STATUS_UNAUTHENTICATED, code: "UNAUTHENTICATED", msg: msg}
}

func ErrNotImplemented(rpc string) error {
	return &apiErr{
		status: tbproto.RpcStatus_RPC_STATUS_INTERNAL,
		code:   "NOT_IMPLEMENTED",
		msg:    rpc + " handler awaiting subsystem",
	}
}

// OkResponse builds an RPC_STATUS_OK response with payload (may be nil).
func OkResponse(payload []byte) *tbproto.RpcResponse {
	return &tbproto.RpcResponse{
		Status:  tbproto.RpcStatus_RPC_STATUS_OK,
		Payload: payload,
	}
}

// ErrToResponse converts any error to an RpcResponse. apiErr maps to its
// declared status/code; other errors map to INTERNAL/INTERNAL with the
// message redacted to "internal error" unless debug is true.
func ErrToResponse(err error, debug bool) *tbproto.RpcResponse {
	if err == nil {
		return OkResponse(nil)
	}
	var ae *apiErr
	if errors.As(err, &ae) {
		return &tbproto.RpcResponse{
			Status: ae.status,
			Error:  &tbproto.RpcError{Code: ae.code, Message: ae.msg},
		}
	}
	msg := "internal error"
	if debug {
		msg = err.Error()
	}
```

```go
package api

import (
	"context"
	"net/netip"
	"time"

	"github.com/rs/zerolog"
	"github.com/token-bay/token-bay/shared/ids"
	tbproto "github.com/token-bay/token-bay/shared/proto"
)

// Deps lists the subsystems an api handler may depend on. Fields left
// nil cause the matching RPCs to register as ErrNotImplemented stubs.
type Deps struct {
	Logger zerolog.Logger
	Now    func() time.Time
	Debug  bool // surface internal-error messages on responses

	Ledger     LedgerService
	Registry   RegistryService
	StunTurn   StunTurnService
	Broker     BrokerService
	Admission  AdmissionService
	Federation FederationService
}

// RequestCtx carries per-call info every handler may need.
type RequestCtx struct {
	PeerID     ids.IdentityID
	RemoteAddr netip.AddrPort
	Now        time.Time
	Logger     zerolog.Logger
}

// handler signature: (ctx, request payload bytes) → response or error.
type handlerFunc func(ctx context.Context, rc *RequestCtx, payload []byte) (*tbproto.RpcResponse, error)

type Router struct {
	deps     Deps
	handlers map[tbproto.RpcMethod]handlerFunc
}

func NewRouter(d Deps) (*Router, error) {
	if d.Now == nil {
		d.Now = time.Now
	}
	r := &Router{deps: d, handlers: map[tbproto.RpcMethod]handlerFunc{}}
	r.handlers[tbproto.RpcMethod_RPC_METHOD_ENROLL] = r.installEnroll()
	r.handlers[tbproto.RpcMethod_RPC_METHOD_BROKER_REQUEST] = r.installBrokerRequest()
	r.handlers[tbproto.RpcMethod_RPC_METHOD_BALANCE] = r.installBalance()
	r.handlers[tbproto.RpcMethod_RPC_METHOD_SETTLE] = r.installSettle()
	r.handlers[tbproto.RpcMethod_RPC_METHOD_USAGE_REPORT] = r.installUsageReport()
	r.handlers[tbproto.RpcMethod_RPC_METHOD_ADVERTISE] = r.installAdvertise()
	r.handlers[tbproto.RpcMethod_RPC_METHOD_TRANSFER_REQUEST] = r.installTransferRequest()
	r.handlers[tbproto.RpcMethod_RPC_METHOD_STUN_ALLOCATE] = r.installStunAllocate()
	r.handlers[tbproto.RpcMethod_RPC_METHOD_TURN_RELAY_OPEN] = r.installTurnRelayOpen()
	return r, nil
}

// Dispatch is the single entry point. It NEVER returns an error: protocol
// errors become *RpcResponse with non-OK status. Transport errors are
// server's concern.
func (r *Router) Dispatch(ctx context.Context, rc *RequestCtx, req *tbproto.RpcRequest) *tbproto.RpcResponse {
	if req.Method == tbproto.RpcMethod_RPC_METHOD_UNSPECIFIED {
		return &tbproto.RpcResponse{
			Status: tbproto.RpcStatus_RPC_STATUS_INVALID,
			Error:  &tbproto.RpcError{Code: "METHOD_ZERO_RESERVED", Message: "method 0 reserved for heartbeat channel"},
		}
	}
	h, ok := r.handlers[req.Method]
	if !ok {
		return &tbproto.RpcResponse{
			Status: tbproto.RpcStatus_RPC_STATUS_INVALID,
			Error:  &tbproto.RpcError{Code: "UNKNOWN_METHOD", Message: req.Method.String()},
		}
	}
	resp, err := h(ctx, rc, req.Payload)
	if err != nil {
		return ErrToResponse(err, r.deps.Debug)
	}
	if resp == nil {
		return OkResponse(nil)
	}
	return resp
}
```

```go
package api

import (
	"context"
	"errors"
	"fmt"

	"google.golang.org/protobuf/proto"

	tbproto "github.com/token-bay/token-bay/shared/proto"
	"github.com/token-bay/token-bay/shared/signing"
)

const enrollStarterGrantCredits uint64 = 1000 // matches admission spec default

type enrollLedger interface {
	IssueStarterGrant(ctx context.Context, identityID []byte, amount uint64) (*tbproto.Entry, error)
}

// enrollAdmission is optional. When nil Deps.Admission, the gate auto-passes.
type enrollAdmission interface {
	Admit(ctx context.Context, identity []byte, accountFingerprint []byte) error
}

// ErrFrozenAdmission lets the admission package surface a frozen state to
// this handler without depending on api. Admission impl returns
// errors.New("frozen") (or sentinel) — we type-check by string match for
// now; a typed error from internal/admission lands when that subsystem is
// rebuilt against api.
//
// Kept as a const so admission tests can match.
const admissionFrozenSentinel = "frozen"

func (r *Router) installEnroll() handlerFunc {
	led, ok := r.deps.Ledger.(enrollLedger)
	if r.deps.Ledger == nil || !ok {
		return notImpl("enroll")
	}
	adm, _ := r.deps.Admission.(enrollAdmission) // nil is OK
	return func(ctx context.Context, rc *RequestCtx, payloadBytes []byte) (*tbproto.RpcResponse, error) {
		var req tbproto.EnrollRequest
		if err := proto.Unmarshal(payloadBytes, &req); err != nil {
			return nil, ErrInvalid("EnrollRequest: " + err.Error())
		}
		if len(req.IdentityPubkey) != 32 {
			return nil, ErrInvalid(fmt.Sprintf("identity_pubkey length %d, want 32", len(req.IdentityPubkey)))
		}
		if len(req.AccountFingerprint) != 32 {
			return nil, ErrInvalid(fmt.Sprintf("account_fingerprint length %d, want 32", len(req.AccountFingerprint)))
		}
		// rc.PeerID derived from mTLS SPKI MUST match the request's pubkey hash;
		// equality is checked at the wire layer, but defense-in-depth here.
		if rc.PeerID != hashPubkey(req.IdentityPubkey) {
			return nil, ErrUnauthenticated("enroll: identity_pubkey does not match mTLS peer")
		}

		if adm != nil {
			if err := adm.Admit(ctx, rc.PeerID[:], req.AccountFingerprint); err != nil {
				if errors.Is(err, errFrozen) || err.Error() == admissionFrozenSentinel {
					return nil, ErrFrozen(err.Error())
				}
				return nil, fmt.Errorf("enroll: admission: %w", err)
			}
		}

		entry, err := led.IssueStarterGrant(ctx, rc.PeerID[:], enrollStarterGrantCredits)
		if err != nil {
			return nil, fmt.Errorf("enroll: starter_grant: %w", err)
		}
		entryBytes, err := signing.DeterministicMarshal(entry)
		if err != nil {
			return nil, fmt.Errorf("enroll: marshal entry: %w", err)
		}

		out := &tbproto.EnrollResponse{
			IdentityId:           rc.PeerID[:],
			StarterGrantCredits:  enrollStarterGrantCredits,
			StarterGrantEntry:    entryBytes,
		}
		respBytes, _ := proto.Marshal(out)
		return OkResponse(respBytes), nil
	}
}

var errFrozen = errors.New(admissionFrozenSentinel)

// hashPubkey computes SHA-256(identity_pubkey) — same as tracker server's
// SPKI-to-IdentityID derivation when the cert wraps that pubkey.
func hashPubkey(pub []byte) idsLike {
	var h idsLike
	sum := sha256Sum(pub)
	copy(h[:], sum[:])
	return h
}

func sha256Sum(b []byte) [32]byte {
	// thin wrapper to keep import list local; declared here so router.go
	// doesn't grow a stray sha256 import.
	return sha256SumImpl(b)
}
```

```go
// ALPN matches the merged plugin trackerclient/internal/idtls.ALPN.
const ALPN = "tokenbay/1"

// ServerCertFromIdentity issues a self-signed X.509 cert wrapping priv —
// mirror of plugin/internal/trackerclient/internal/idtls.CertFromIdentity.
func ServerCertFromIdentity(priv ed25519.PrivateKey) (tls.Certificate, error) {
	if len(priv) != ed25519.PrivateKeySize {
		return tls.Certificate{}, fmt.Errorf("server: priv length %d, want %d",
			len(priv), ed25519.PrivateKeySize)
	}
	tmpl := &x509.Certificate{
		SerialNumber:          big.NewInt(1),
		Subject:               pkix.Name{CommonName: "token-bay-tracker"},
		NotBefore:             time.Unix(0, 0),
		NotAfter:              time.Date(9999, 12, 31, 0, 0, 0, 0, time.UTC),
		KeyUsage:              x509.KeyUsageDigitalSignature,
		ExtKeyUsage:           []x509.ExtKeyUsage{x509.ExtKeyUsageServerAuth, x509.ExtKeyUsageClientAuth},
		BasicConstraintsValid: true,
	}
	der, err := x509.CreateCertificate(rand.Reader, tmpl, tmpl, priv.Public(), priv)
	if err != nil {
		return tls.Certificate{}, fmt.Errorf("server: CreateCertificate: %w", err)
	}
	return tls.Certificate{Certificate: [][]byte{der}, PrivateKey: priv}, nil
}

// SPKIToIdentityID hashes the cert's SubjectPublicKeyInfo. Empty SPKI is
// rejected to defeat hand-crafted truncated certs.
func SPKIToIdentityID(cert *x509.Certificate) (ids.IdentityID, error) {
	if cert == nil {
		return ids.IdentityID{}, errors.New("server: nil cert")
	}
	if len(cert.RawSubjectPublicKeyInfo) == 0 {
		return ids.IdentityID{}, errors.New("server: empty SPKI")
	}
	var id ids.IdentityID
	sum := sha256.Sum256(cert.RawSubjectPublicKeyInfo)
	copy(id[:], sum[:])
	return id, nil
}

// VerifyClientCert is the VerifyPeerCertificate hook for the server's TLS
// config. Accepts exactly one Ed25519 client cert with non-empty SPKI.
func VerifyClientCert(rawCerts [][]byte) error {
	if len(rawCerts) == 0 {
		return errors.New("server: no peer certificate")
	}
	if len(rawCerts) > 1 {
		return errors.New("server: peer offered more than one cert")
	}
	parsed, err := x509.ParseCertificate(rawCerts[0])
	if err != nil {
		return fmt.Errorf("server: parse peer cert: %w", err)
	}
	if _, ok := parsed.PublicKey.(ed25519.PublicKey); !ok {
		return errors.New("server: peer cert is not Ed25519")
	}
	if len(parsed.RawSubjectPublicKeyInfo) == 0 {
		return errors.New("server: empty SPKI")
	}
	return nil
}

// MakeServerTLSConfig builds the tls.Config used by the QUIC listener.
// Mutual auth: requires any client cert and validates via VerifyClientCert.
func MakeServerTLSConfig(ourCert tls.Certificate) *tls.Config {
	return &tls.Config{
		Certificates:           []tls.Certificate{ourCert},
		ClientAuth:             tls.RequireAnyClientCert,
		InsecureSkipVerify:     true, //nolint:gosec // see VerifyPeerCertificate
		VerifyPeerCertificate: func(rawCerts [][]byte, _ [][]*x509.Certificate) error {
			return VerifyClientCert(rawCerts)
		},
		NextProtos:             []string{ALPN},
		MinVersion:             tls.VersionTLS13,
		SessionTicketsDisabled: true,
	}
}
```

```go
import (
	"context"
	"net/netip"
	"sync"

	quicgo "github.com/quic-go/quic-go"
	"github.com/token-bay/token-bay/shared/ids"

// quicConn is the slice of quic-go.Conn we use; declared as an interface
// so connection_test can swap a fake.
type quicConn interface {
	OpenStreamSync(ctx context.Context) (*quicgo.Stream, error)
	AcceptStream(ctx context.Context) (*quicgo.Stream, error)
	Context() context.Context
	CloseWithError(quicgo.ApplicationErrorCode, string) error
}

// Connection is the per-peer server-side state.
type Connection struct {
	peerID     ids.IdentityID
	remoteAddr netip.AddrPort
	conn       quicConn

	ctx    context.Context
	cancel context.CancelFunc

	// streamSeq tracks how many client-initiated streams we've accepted.
	// Stream #1 is the heartbeat; #2+ are RPC streams.
	streamSeq atomic.Int64

	closeOnce sync.Once
}

func newConnection(parent context.Context, qc quicConn, peerID ids.IdentityID, addr netip.AddrPort) *Connection {
	ctx, cancel := context.WithCancel(parent)
	return &Connection{
		peerID: peerID, remoteAddr: addr, conn: qc, ctx: ctx, cancel: cancel,
	}
}

// PeerID, RemoteAddr accessors used by handlers via RequestCtx.
func (c *Connection) PeerID() ids.IdentityID  { return c.peerID }
func (c *Connection) RemoteAddr() netip.AddrPort { return c.remoteAddr }

// Close cancels the connection context and tears down the QUIC conn.
// Idempotent.
func (c *Connection) Close() {
	c.closeOnce.Do(func() {
		c.cancel()
		_ = c.conn.CloseWithError(0, "server close")
	})
}
```

```go
package server

import (
	"context"
	"errors"
	"io"
	"time"

	tbproto "github.com/token-bay/token-bay/shared/proto"
)

type heartbeatRegistry interface {
	Heartbeat(id idsLike, now time.Time) error
}

// Internal alias to avoid leaking ids.IdentityID into every signature here.
type idsLike = ids.IdentityID  // declared once in framing.go or connection.go

// runHeartbeat reads bare HeartbeatPing frames off stream and writes
// HeartbeatPong replies. Non-blocking against the connection's other
// streams; exits when the stream EOFs or ctx cancels.
func runHeartbeat(ctx context.Context, stream io.ReadWriter, peerID ids.IdentityID, reg heartbeatRegistry, now func() time.Time, maxFrameSize int, log zerolog.Logger) {
	for {
		if ctx.Err() != nil {
			return
		}
		var ping tbproto.HeartbeatPing
		err := ReadFrame(stream, &ping, maxFrameSize)
		if errors.Is(err, io.EOF) {
			return
		}
		if err != nil {
			log.Warn().Err(err).Msg("server: heartbeat read")
			return
		}
		_ = reg.Heartbeat(peerID, now())
		if err := WriteFrame(stream, &tbproto.HeartbeatPong{Seq: ping.Seq}, maxFrameSize); err != nil {
			log.Warn().Err(err).Msg("server: heartbeat write")
			return
		}
	}
}
```

```go
package server

import (
	"context"
	"errors"
	"io"

	tbproto "github.com/token-bay/token-bay/shared/proto"
	"github.com/token-bay/token-bay/tracker/internal/api"
)

// handleRPCStream reads one framed RpcRequest, dispatches it via the
// Router, writes one framed RpcResponse, and closes the write side.
// Increments the connection's stream counter for diagnostic purposes.
func (s *Server) handleRPCStream(ctx context.Context, stream rpcStream, c *Connection) {
	defer stream.Close()
	s.wg.Add(1)
	defer s.wg.Done()

	var req tbproto.RpcRequest
	maxFrame := s.deps.Config.Server.MaxFrameSize
	if err := ReadFrame(stream, &req, maxFrame); err != nil {
		// Truncated / oversize / garbage → write INVALID if we can, else just close.
		if errors.Is(err, io.EOF) || errors.Is(err, io.ErrUnexpectedEOF) {
			return
		}
		_ = WriteFrame(stream, &tbproto.RpcResponse{
			Status: tbproto.RpcStatus_RPC_STATUS_INVALID,
			Error:  &tbproto.RpcError{Code: "FRAME", Message: err.Error()},
		}, maxFrame)
		return
	}
	rc := &api.RequestCtx{
		PeerID:     c.peerID,
		RemoteAddr: c.remoteAddr,
		Now:        s.deps.Now(),
		Logger:     s.deps.Logger.With().Str("peer", c.peerID.String()).Logger(),
	}
	resp := func() (out *tbproto.RpcResponse) {
		defer func() {
			if r := recover(); r != nil {
				s.deps.Logger.Error().Interface("panic", r).Msg("server: handler panic")
				out = &tbproto.RpcResponse{
					Status: tbproto.RpcStatus_RPC_STATUS_INTERNAL,
					Error:  &tbproto.RpcError{Code: "PANIC", Message: "handler panicked"},
				}
			}
		}()
		return s.deps.API.Dispatch(ctx, rc, &req)
	}()
	if err := WriteFrame(stream, resp, maxFrame); err != nil {
		s.deps.Logger.Warn().Err(err).Msg("server: write response")
	}
	_ = stream.CloseWrite()
}

// rpcStream is the slice of *quicgo.Stream the handler uses; satisfied by
// quic-go and by net.Pipe for tests.
type rpcStream interface {
	io.Reader
	io.Writer
	Close() error
	CloseWrite() error
}
```

```go
package server

import (
	"context"
	"time"

	tbproto "github.com/token-bay/token-bay/shared/proto"
)

func (s *Server) PushOfferTo(id idsLike, push *tbproto.OfferPush) (<-chan *tbproto.OfferDecision, bool) {
	s.mu.Lock()
	if !s.running {
		s.mu.Unlock()
		return nil, false
	}
	c, ok := s.connsByPeer[id]
	s.mu.Unlock()
	if !ok {
		return nil, false
	}
	out := make(chan *tbproto.OfferDecision, 1)
	go s.runOfferPush(c, push, out)
	return out, true
}

func (s *Server) runOfferPush(c *Connection, push *tbproto.OfferPush, out chan<- *tbproto.OfferDecision) {
	defer close(out)
	maxFrame := s.deps.Config.Server.MaxFrameSize
	timeout := time.Duration(s.deps.Config.Broker.OfferTimeoutMs) * time.Millisecond
	if timeout <= 0 {
		timeout = 5 * time.Second
	}
	ctx, cancel := context.WithTimeout(c.ctx, timeout)
	defer cancel()

	stream, err := c.conn.OpenStreamSync(ctx)
	if err != nil {
		out <- &tbproto.OfferDecision{Accept: false, RejectReason: "connection_lost"}
		return
	}
	defer stream.Close()
	if _, err := stream.Write([]byte{api.PushTagOffer}); err != nil {
		out <- &tbproto.OfferDecision{Accept: false, RejectReason: "connection_lost"}
		return
	}
	if err := WriteFrame(stream, push, maxFrame); err != nil {
		out <- &tbproto.OfferDecision{Accept: false, RejectReason: "connection_lost"}
		return
	}
	if err := stream.CloseWrite(); err != nil {
		out <- &tbproto.OfferDecision{Accept: false, RejectReason: "connection_lost"}
		return
	}
	var dec tbproto.OfferDecision
	if err := ReadFrame(stream, &dec, maxFrame); err != nil {
		if ctx.Err() == context.DeadlineExceeded {
			out <- &tbproto.OfferDecision{Accept: false, RejectReason: "timeout"}
		} else {
			out <- &tbproto.OfferDecision{Accept: false, RejectReason: "connection_lost"}
		}
		return
	}
	out <- &dec
}
```

```go
func (s *Server) Shutdown(ctx context.Context) error {
	s.mu.Lock()
	if !s.running {
		s.mu.Unlock()
		return nil
	}
	s.running = false
	cancel := s.serverCancel
	s.mu.Unlock()

	cancel() // cancels serverCtx → all derived contexts → goroutines exit

	done := make(chan struct{})
	go func() {
		s.wg.Wait()
		close(done)
	}()
	select {
	case <-done:
		return nil
	case <-ctx.Done():
		s.mu.Lock()
		for _, c := range s.connsByPeer {
			c.Close()
		}
		s.mu.Unlock()
		return ctx.Err()
	}
}
```

The push-cancel semantics fall out for free: `runOfferPush` derives its ctx from `c.ctx` which is canceled when `serverCtx` is canceled, so the push goroutine's `OpenStreamSync` / `ReadFrame` returns; the `out` channel is closed via the `defer close(out)`, which is the "drain reject" signal per spec §10.3.4 ("ackCh closed without value").

```go
				return reportConfigError(cmd, err)
			}

			}

			})

			}

				// stunturn.Reflect is a STUN binding-response builder, not an
				// address translator; the API handler only needs the observed
				// remote addr (the QUIC layer's view), so the simplest correct
				// choice is identity for v1. NAT detection is a broker-era
```
